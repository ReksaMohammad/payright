Apakah Data Saya Aman Dengan PayRight?

Keamanan Anda adalah prioritas utama kami. Oleh karenanya, kami selalu berusaha untuk melindungi informasi pribadi Anda seoptimal mungkin.

Keamanan setara bank dengan Security Socket Layer (SSL) 256-bit

Kami menggunakan enkripsi SSL 256-bit, teknologi yang sama yang digunakan bank-bank untuk menjaga agar informasi akun Anda aman.

Penyimpanan data

Data Anda tersimpan di server Amazon Web Services (AWS), yang merupakan Cloud Infrastructure as a Service paling dipercaya di dunia..

Backup data

Data Anda di-backup ke beberapa server dengan lokasi yang berbeda-beda. Backup data dilakukan rutin, selama 1x24 jam dan 7 hari dalam seminggu.

Enkripsi data

Kami menerapkan enkripsi terhadap data-data berikut agar tidak dapat dibaca oleh siapapun kecuali oleh pemilik akun yang bersangkutan.

Log perubahan data

Untuk memastikan keamanan data Anda, Payright System juga menyimpan catatan (log) untuk semua akses ke akun Anda.

SISTEM BERBASIS WEB

Sistem yang kami gunakan adalah berbasis website, sehingga Anda tidak perlu repot - repot menginstall software atau aplikasi pada komputer Anda. Cukup akses melalui website yang telah kami sediakan dan gunakan akun Anda.

BEBAN BIAYA FLEKSIBEL SESUAI KOMPLEKSITAS

Tidak seperti software penggajian biasanya dalam penetapan skema harga, kami memberikan biaya tidak berdasarkan pada besarnya jumlah karyawan Anda. Namun, kami menggunakan kompleksitas sistem yang Anda gunakan. Dengan menggunakan Star System Score, kami menilai kerumitan sistem yang Anda gunakan untuk diterapkan di perusahaan Anda.

Apa itu Star System Score? Selengkapnya Unduh Disini.

VISUALISASI DATA

Kami memberikan data secara visual kepada Anda agar data yang Anda lihat lebih jelas dan detil. Data tersebut termasuk data keseluruhan dari setiap karyawan yang Anda miliki..

PIHAK KETIGA

Sistem kami adalah sebagai pihak ketiga dari Anda, sehingga tidak akan terjadi kecurangan dalam data Anda seperti absensi karyawan dan lain - lain.
